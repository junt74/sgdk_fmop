# 画面レイアウト仕様

## パラメータ名と数値の桁数対応

MDSDRV-mml の音色定義コメント行（`;AR5 DR5 SR5 RR4 SL4 TL7  K2 ML4 DT2 SSG`）には、
パラメータ名とビット幅が示されている。
画面表示では **パラメータ名の列幅と、その下に並ぶ数値の最大桁数を一致させる** こと。

### グローバルパラメータ（音色全体）

`@0 fm 3 0` の末尾2値はアルゴリズム（ALG）とフィードバック（FB）であり、**編集対象のパラメータとして扱う**。
OP別パラメータとは区別して画面上部などに独立表示し、カーソルで選択・変更できること。

| パラメータ名 | ビット幅 | 値の範囲 | 最大桁数 | 表示幅（文字数） |
|-------------|---------|---------|---------|----------------|
| ALG         | 3       | 0〜7    | 1桁     | 3              |
| FB          | 3       | 0〜7    | 1桁     | 2              |

### OPパラメータ一覧（OP1〜OP4 共通）

| パラメータ名 | ビット幅 | 値の範囲 | 最大桁数 | 表示幅（文字数） |
|-------------|---------|---------|---------|----------------|
| AR          | 5       | 0〜31   | 2桁     | 2              |
| DR          | 5       | 0〜31   | 2桁     | 2              |
| SR          | 5       | 0〜31   | 2桁     | 2              |
| RR          | 4       | 0〜15   | 2桁     | 2              |
| SL          | 4       | 0〜15   | 2桁     | 2              |
| TL          | 7       | 0〜127  | 3桁     | 3              |
| K           | 2       | 0〜3    | 1桁     | 1              |
| ML          | 4       | 0〜15   | 2桁     | 2              |
| DT          | 2 ※1   | 0〜7    | 1桁     | 1              |
| SSG         | 4       | 0〜15   | 2桁     | 3 ※2          |

※1 DT はMDSDRVでは符号なし3ビット（0〜7）で扱われる場合がある。実装時に要確認。
※2 "SSG" はラベルが3文字のため、表示幅はラベルに合わせて3文字とする。

### 留意事項

- **ラベル行と数値行の列位置を完全に揃えること。**
  ラベルが2文字のパラメータ（AR, DR, SR, RR, SL, ML）は2文字分の列幅を確保し、
  数値は右詰めまたはゼロ埋めで表示する。

- **TL のみ3桁** であるため、隣接するパラメータとの区切りスペースに注意すること。
  MMLコメント行でも `TL7` の前後にスペースが多めに入っているのはそのため。

- **K, DT は1桁** であり、ラベルも1〜2文字と短い。
  他のパラメータと並べる際、列が詰まりすぎないよう最低1文字の余白を設ける。

- カーソルはパラメータ名の真下（数値行と同じ列）に表示する。
  ラベル幅と数値幅が揃っていないと、カーソル位置がずれて見えるため必ず一致させること。

### 表示イメージ（参考）

```
ALG FB
  3  0

AR DR SR RR SL  TL K ML DT SSG
31  0 19  5  0  23 0  0  0   0   ← OP1
31  6  0  4  3  19 0  0  0   0   ← OP2
31 15  0  5  4  38 0  4  0   0   ← OP3
31 27  0 11  1   0 0  1  0   0   ← OP4
      ^                           ← カーソル例（SR列）
```

ALG/FB もカーソルで選択・変更できる。カーソルがALG/FB領域とOP領域を行き来できるよう設計すること。

数値は右詰め表示とし、ラベル幅に満たない場合は先頭をスペースで埋める。
