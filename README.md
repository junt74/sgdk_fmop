# sgdk_fmop

MDSDRVの音色（FM音源パラメータ）をメガドライブ実機上でリアルタイムに編集するためのツールプログラムです。

**成果物はセガメガドライブ（海外版: Sega Genesis）向けの ROM イメージ**（`.bin`）です。ビルドは `out/FMOP.bin` を生成し、エミュレータまたは通常のメガドライブ向け書き込み手順で利用できます。PC用の単体実行ファイルではありません。

**本リポジトリのソフトウェアは現在も実装中であり、完成品・製品版ではありません。** 仕様・操作・表示は予告なく変わる可能性があります。

## 概要

画面に表示される数値は **MDSDRV-mml の音色定義（`@n fm …` 以下の OP 行と、同じ並びの ALG・FB）** に対応しています。
気に入った音色ができたら、画面の値をそのまま PC の MML に転記し、`@` 行の ALG・FB と OP 4 行を合わせれば同じデータになります。

## MDSDRV-mml 音色定義との対応

MDSDRV-mml の音色定義は以下の形式です。

```
;==== Instrument section =====

@0 fm 3 0 ;Instrument

;AR5 DR5 SR5 RR4 SL4 TL7  K2 ML4 DT2 SSG
  31   0  19   5   0  23   0   0   0   0
  31   6   0   4   3  19   0   0   0   0
  31  15   0   5   4  38   0   4   0   0
  31  27   0  11   1   0   0   1   0   0
```

### 各フィールドの説明

| フィールド | 意味 |
|-----------|------|
| `@n` / `fm` | MML の音色番号・FM 指定。本ツールの画面には **数値としては出さず**、PC 側の `@` 行を自分で合わせる |
| `ALG` / `FB` | アルゴリズム・フィードバック。**編集可能**（画面上段の数値。MML ヘッダの 3 0 に相当する部分） |
| `AR` | Attack Rate（アタックレート） |
| `DR` | Decay Rate（ディケイレート） |
| `SR` | Sustain Rate（サステインレート） |
| `RR` | Release Rate（リリースレート） |
| `SL` | Sustain Level（サステインレベル） |
| `TL` | Total Level（トータルレベル） |
| `K` | Key Scale（キースケール） |
| `ML` | Multiple（周波数倍率） |
| `DT` | Detune（デチューン） |
| `SSG` | SSG-EG。**画面では 3 桁表示のうち「100 の位」と「下 2 桁」が別セル**（カーソルも 2 列）。転記時はこの 3 桁を MML の SSG 列として読み替える |

OP 行 4 行はオペレータ OP1〜OP4 に対応します（上から順）。

## 操作方法

**D-pad のカーソル移動は、方向キーを押した瞬間（エッジ）だけ反応し、押し続けても連続移動しません。** **C ＋十字**による数値変更は、初回のあとオートリピートがあります。

### B を押していないとき（パラメータ編集）

| 操作 | 動作 |
|------|------|
| D-pad 左右（エッジ） | **ALG / FB 行**: ALG と FB の選択を切り替え。**OP 表の行にいるとき**: 左右のパラメータ列へ移動 |
| D-pad 下（エッジ） | **ALG / FB 行から**: OP 表の先頭行へ入る。**OP 表内**: 下の OP 行へ |
| D-pad 上（エッジ） | **OP 表の 2 行目以降**: 上の OP へ。**OP1 の行**: ALG / FB 行へ戻る |
| C ＋ 十字（単独方向・斜め不可） | 選択セルの数値を変更（**右 +1 / 左 −1 / 上 +10 / 下 −10**、最小・最大でクランプ）。初回のあと **約 20 フレーム待ってから約 10 フレーム間隔**でリピート |
| A ボタン（押下中） | 現在のパッチで発音試聴 |
| A ボタン（離す） | 消音 |

### B を押しているとき（確認ノートのみ）

**B 押下中はカーソル移動も C によるパラメータ変更も行われません。** カーソルスプライトだけが `NOTE:` の値の位置に表示されます。

| 操作 | 動作 |
|------|------|
| B ＋ 左右（B 押下中に左右を**新規に**押す） | 確認ノート（`NOTE:` 行）を **半音** ずつ変更（**o0 C〜o8 B** でクランプ） |
| B ＋ 上下（B 押下中に上下を**新規に**押す） | 確認ノートを **1 オクターブ**（±12 半音）ずつ変更 |

確認用ノートは A 試聴時の MIDI 基準音程に使われます（既定は **o4 C**）。

## ビルド方法

`make release` の結果、**メガドライブ用 ROM** `out/FMOP.bin` が出力されます。

### macOS

```sh
# GDK 環境変数が未設定の場合は /Users/junt74/SGDK を自動使用
make release

# ビルド後にエミュレータ（OpenEMU）で起動
make run
```

### Windows

```bat
set GDK=C:\LIBRARY\SGDK
make release

# エミュレータ指定例
make run EMU="C:/path/to/Fusion.exe"
```

### 前提条件

- [SGDK](https://github.com/Stephane-D/SGDK)（環境変数 `GDK` で指定。**Windows では未設定だと makefile がエラー**になります）
- m68k-elf-gcc（**Windows では SGDK に同梱のツールチェーンを使うのが一般的**。macOS 等では `brew install m68k-elf-gcc` など）
- Java（ResComp 用。**macOS** で Homebrew の OpenJDK を使うと Makefile が `java` を明示できる場合があります。Windows では SGDK 同梱・またはインストール済みの Java をパスで利用）
